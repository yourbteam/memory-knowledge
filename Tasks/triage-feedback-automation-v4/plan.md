## Scope

Prepare the V4 enhancement program as a set of concrete future tasks and align the roadmap to those slices.

## Execution Steps

1. Add a future roadmap section for V4 feedback automation.
2. Create one task folder per V4 slice.
3. Record the practical problem, intended upgrade, and likely tool/data surfaces for each slice.
4. Keep the slices ordered for recommended delivery.

## Deliverables

- updated `docs/roadmap.md`
- task folders for each V4 slice with:
  - `analysis.md`
  - `plan.md`

## Validation

- confirm the roadmap entries match the task folder set
- confirm each slice is detailed enough to implement later without rediscovering the objective
